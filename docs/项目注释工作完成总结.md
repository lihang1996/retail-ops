## 核心业务流程

### 1. 订单履约

```text
导入订单 (pending_payment)
  → 支付 POST /api/proj/order/pay：选仓 + 锁库存 + 写 warehouse_id (paid)
  → 分仓 POST /api/proj/order/allocate 或创建发货单时自动分仓 (allocated)
  → 生成发货单 → 拣货 → 出库 (shipped)
```

AI playbook：`order-fulfillment` · 详见 M4 / M7 文档。

### 2. 商品上架

```text
draft → pending_review → on_sale ↔ off_sale
         ↑ 审批待办通过
```

AI playbook：`product-onboarding`、`approval-flow`

### 3. 演示黄金链路（新人）

```text
经营总览 → 商品/入库 → 履约 → 3D 仓库 → AI 业务助手提问
```

AI playbook：`demo-golden-path`

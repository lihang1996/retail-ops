## 商品状态流转

```
┌─────────┐
│  draft  │ 草稿（初始状态）
│  草稿   │ - 可编辑商品信息
└────┬────┘ - 可增删改 SKU
     │      - 可删除商品
     │ submitReview()
     ▼
┌──────────────┐
│pending_review│ 待审核
│   待审核     │ - 已提交审批
└────┬─────────┘ - 等待审批人审批
     │
     │ approval.approve()
     ▼
┌──────────┐
│ on_sale  │ 在售
│   在售   │ - 不可编辑商品
└────┬─────┘ - 不可增删改 SKU
     │      - 可下架
     │ offSale()
     ▼
┌──────────┐
│ off_sale │ 下架
│   下架   │ - 可重新提交审核
└──────────┘ - 可编辑商品信息
             - 可增删改 SKU
```

## 工厂模式详解

### dict-service 工厂

**目的**：消除字典表 CRUD 的重复代码

**配置参数**：
```javascript
{
  table: 'brands',           // 数据库表名
  idField: 'brand_id',       // 主键字段名
  nameField: 'brand_name',   // 名称字段名
  auditPrefix: 'product:brand', // 审计日志前缀
  label: '品牌',             // 中文标签（用于错误提示）
}
```

**生成的方法**：
1. `list(ctx, query)` - 列表查询
   - 租户隔离
   - 状态筛选
   - 名称模糊搜索
2. `get(ctx, query)` - 详情查询
   - 租户归属校验
3. `create(ctx, body)` - 创建
   - 名称唯一性校验
   - 审计日志记录
   - 类目表支持 parent_id
4. `update(ctx, body)` - 更新
   - 租户归属校验
   - 审计日志记录

## 业务规则总结

### 1. 商品状态控制

```javascript
// 已上架商品的限制
if (product.status === 'on_sale') {
  // ❌ 不可编辑商品信息
  // ❌ 不可创建 SKU
  // ❌ 不可删除 SKU
  // ✅ 可以下架
}

// 草稿商品
if (product.status === 'draft') {
  // ✅ 可以编辑所有信息
  // ✅ 可以增删改 SKU
  // ✅ 可以删除商品
  // ✅ 可以提交审核
}
```

### 2. 上架条件校验

```javascript
async _validateOnSale(db, tenantId, product) {
  // 1. 必须选择类目
  if (!product.category_id) bizError('上架前需选择类目')
  
  // 2. 必须设置主图
  if (!product.main_image) bizError('上架前需设置主图')
  
  // 3. 至少一个有效 SKU
  const skus = await db('product_skus')
    .where({ tenant_id, product_id, status: 'active' })
  if (!skus.length) bizError('上架前需至少一个有效 SKU')
  
  // 4. SKU 需设置售价
  const hasPrice = skus.some((s) => Number(s.sale_price) > 0)
  if (!hasPrice) bizError('上架前 SKU 需设置售价')
}
```

### 3. 租户隔离

```javascript
// 所有查询都带 tenant_id
db('products').where({ tenant_id })

// 名称唯一性校验限定在租户内
const exists = await db('brands')
  .where({ tenant_id, brand_name })
  .first()
```

### 4. 审计日志

## 库存风险监控

```javascript
// product.js service 层计算
const stockRows = await db('product_skus as sku')
  .leftJoin('stocks as s', 'sku.sku_id', 's.sku_id')
  .select(
    'sku.product_id',
    db.raw('SUM(s.available_qty) as stock_total'),
    // 统计低库存 SKU 数量
    db.raw(`SUM(CASE 
      WHEN s.available_qty <= s.warning_qty 
      THEN 1 ELSE 0 
    END) as risk_sku_count`)
  )

// 风险等级判断
let stockRisk = 'none'
if (stock_total > 0) {
  stockRisk = risk_sku_count > 0 ? 'has_risk' : 'normal'
}
```

**风险等级**：
- `none` - 无库存
- `has_risk` - 有低库存 SKU（available_qty <= warning_qty）
- `normal` - 库存正常

    }
  }
}

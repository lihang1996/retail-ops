## 🛠 开发指南

### 使用设计令牌

```css
/* 推荐 */
.my-card {
  padding: var(--spacing-6);
  border-radius: var(--radius-lg);
  box-shadow: var(--shadow-sm);
  color: var(--color-gray-900);
  font-size: var(--text-base);
}

/* 不推荐 */
.my-card {
  padding: 24px;  /* 应该用 var(--spacing-6) */
  border-radius: 14px;  /* 应该用 var(--radius-lg) */
  color: #111827;  /* 应该用 var(--color-gray-900) */
}
```

### 添加悬停效果

```css
.interactive-card {
  transition: all var(--transition-base);
  cursor: pointer;
}

.interactive-card:hover {
  transform: translateY(-2px);
  box-shadow: var(--shadow-lg);
  border-color: var(--color-primary-300);
}
```

### 使用工具类

```html
<!-- 文字颜色 -->
<span class="text-primary">主色</span>
<span class="text-success">成功</span>
<span class="text-muted">次要信息</span>

<!-- 过渡动画 -->
<div class="transition hover-lift">悬停抬升</div>

<!-- 字体 -->
<span class="font-mono">1,234.56</span>
```

## 💡 设计哲学

### 简洁但不简单
每个元素都有其存在的意义，去除装饰性设计，保留功能性美感。

### 一致但不单调
统一的设计语言，但通过细节变化避免视觉疲劳。

### 专业但不冰冷
B2B 平台的专业感，配合温暖的配色和流畅的动效。

### 高效但不匆忙
减少操作步骤，但保留必要的视觉反馈和过渡动画。

# M6: 扩展模块 - 项目介绍

> 更新时间：2026-06-26  
> 模块状态：✅ 已实现（审批/审计完整；客户/财务/营销为薄模块只读）

## 模块概览

**M6: 扩展模块** 在核心零售链路旁，提供 **审批流、审计追溯、运营工作台**，以及 **客户 / 财务 / 营销** 三个演示级只读视图，用于丰富 Demo 场景，但不承担完整业财闭环。

## M6-1 审批中心

### 业务流程

```text
运营提交商品上架审批
  → approvals.status = pending
  → 审批人打开「审批待办」
  → 通过：调用 product.onSale，商品 → on_sale
  → 驳回：商品退回 draft，review_status = rejected
```

### 关键文件

- `app/service/approval.js` — 提交、待办列表、通过/驳回
- `app/controller/approval.js`
- `app/pages/dashboard/approval/approval-todo.vue`

### 主要 API

```text
POST /api/proj/approval/submit      # 提交审批（approval:submit）
GET  /api/proj/approval/todo_list     # 待办列表
POST /api/proj/approval/approve       # 通过
POST /api/proj/approval/reject        # 驳回
```

### 权限

- `approval:submit` — 提交
- `approval:view` — 查看
- `approval:approve` — 审批通过/驳回

AI playbook：[approval-flow.js](../app/knowledge/playbooks/approval-flow.js)（上架审批在哪里处理？）

## M6-2 审计日志

### 职责

记录登录、商品变更、入库出库、审批等 **action_code**，支持按操作人、对象、时间筛选。

### 关键文件

- `app/service/audit.js`
- `app/pages/dashboard/audit/audit-log.vue`
- `app/pages/dashboard/common/audit-action-dict.js` — 前端动作文案映射

### 主要 API

```text
GET /api/proj/audit/list
```

经营总览可展示最近审计动态（`OverviewAuditFeed`）。

---

## M6-3 运营工作台

### 职责

`workbench.js` 聚合多模块摘要（履约待办、库存风险、客户分段、营销即将结束等），供 `ops-console.vue` 与部分列表页 KPI 使用。

### 主要 API

```text
GET /api/proj/workbench/ops
GET /api/proj/workbench/fulfillment
```

---

## M6-4 客户中心（只读）

### 数据来源

客户档案由 **历史订单** 聚合：姓名、手机（脱敏）、订单数、累计消费、客户分段（新客/活跃/复购/VIP）。

### 边界

- ❌ 不支持手动建档、编辑客户
- ✅ 列表筛选、分页、分段统计
- 权限：`customer:view`
- 手机号：`customer:phone:view` 控制是否脱敏；**仅 admin 角色**在 seed 中拥有该权限，其他角色见脱敏号码

### 主要 API

```text
GET /api/proj/customer/list    # 分页列表（含分段统计字段）
GET /api/proj/customer         # 详情（query: customer_id）
```

客户列表页 KPI 摘要来自 `GET /api/proj/workbench/ops` 的 `customerSummary`，**无**独立 `/customer/summary` 路由。

---

## M6-5 财务分析（只读）

### 口径

- 收入：已支付 / 已分仓 / 已发货等有效订单
- 指标：订单收入、净收入、客单价、退款金额与退款率
- 维度：近 7 天趋势、渠道占比、支付方式分布

### 边界

- ❌ 无凭证、对账、开票写操作
- ✅ 刷新报表、图表展示

### 主要 API

```text
GET /api/proj/finance/summary
```

---

## M6-6 营销活动（只读）

### 能力

- 活动列表（进行中 / 待开始 / 已结束）
- 点击活动查看关联商品与促销价
- 运营待办：即将结束活动计数

### 边界

- ❌ 不支持创建/编辑活动
- ✅ 列表、详情只读

### 主要 API

```text
GET /api/proj/marketing/activity/list   # 活动列表
GET /api/proj/marketing/activity        # 活动详情（query 传 activity_id）
```

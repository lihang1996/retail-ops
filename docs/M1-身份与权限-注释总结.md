## 核心安全机制

### 1. JWT 双重校验
```javascript
// auth.js 中间件
1. 验证 JWT 签名和过期时间
2. 查询 login_sessions 表确认会话未撤销
→ 防止 logout 后的 token 继续使用
```

### 2. 租户隔离
```javascript
// 所有查询/写入都带 tenant_id 条件
const list = await db('orders')
  .where({ tenant_id: tenantId })
  .select('*')

// assertRowInTenant() 防止跨租户访问
await assertRowInTenant(db, 'orders', tenantId, 'order_id', orderId, '订单')
```

### 3. RBAC 权限控制
```javascript
// permission.js 中间件
user → user_roles → role_permissions → permissions
// 关键：role.tenant_id 必须匹配当前租户
.andWhere('r.tenant_id', tenantId)
```

### 4. 租户状态管控
```javascript
// tenant-context.js 中间件
- disabled: 拒绝所有请求
- frozen/overdue: 禁止写操作（但允许 logout）
- active: 无限制
```

## 权限码设计

### 命名规范
```
menu:{module}              - 菜单权限
{module}:view              - 查看权限
{module}:create            - 创建权限
{module}:update            - 更新权限
{module}:delete            - 删除权限
{module}:{specific_action} - 特定操作权限
```

### 示例
```javascript
'menu:overview'          // 经营总览菜单
'product:view'           // 查看商品
'product:submit_review'  // 提交上架审核
'stock:inbound'          // 入库操作
'order:pay'              // 支付确认
'shipment:pick'          // 拣货操作
```
